Цель

Разработать приложение с веб-интерфейсом для сбора, анализа и многофункциональной обработки новостей из мира криптовалют

Задачи

Углубиться в изучении обработки естественного языка на python.

Изучить современные средства для парсинга сайтов, написать программы-прототипы для сбора новостей с одного английского и одного русского ресурсов.

Научиться работать с базами данных и использовать их в проекте.

Разработать веб-интерфейс на основе современных технологий и интегрировать его с основной программой.

Методы исследования

Работа будет выполнена на персональном компьютере под управлением ОС Windows 10. В качестве основного языка программирования для сервера выбран Python. Для клиента - JavaScript. Также, на сервере будут использованы библиотеки NLTK, beautifulSoap и Requests. А клиент будет написан с использованием фреймворков VueJS и Quasar, а также библиотеки маршрутизации Vue Router и библиотеки HTTP-запросов Axios.
